Dedicated Workspace

專屬團隊獨立系統

不是提供一套共用系統，而是為每一個政治服務團隊打造專屬的智慧服務管理平台。

專屬登入網址

每個團隊可擁有自己的入口、品牌名稱、Logo 與主色系。

資料完全隔離

不同服務團隊之間資料互不共享，降低資料外洩與誤用風險。

角色權限控管

主任、助理、外勤、檢視者可分層管理，控制資料查看與操作範圍。

專屬 AI 助理

可依服務團隊的案件分類、常見承辦單位與行政區域進行客製。

多服務處協作

總服務處、地方服務站、議會研究室可共同追蹤案件進度。

可擴充 SaaS

未來可複製給不同議員、立委、村里長或地方服務團隊使用。

System Settings

系統設定

串接 Google Sheet、Apps Script、Gemini API 與未來 LINE 官方帳號。

API 設定

請把 Apps Script 部署後的 Web App URL 填入 app.js 的 API_URL。 

const API_URL = "你的 Google Apps Script Web App URL";

Gemini Key 目前可先空白。之後到 Google Sheet 的 Settings 分頁填入 GEMINI_API_KEY，並把 AI_ENABLED 改成 true。
